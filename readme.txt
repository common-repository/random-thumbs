Este plugin exibe alguns posts de forma aleatória no website.

== Description ==

Este plugin cria blocos com o conteúdo de alguns posts capturados aleatoriamente e os exibe também aleatoriamente.
Uma forma de chamar a atenção dos seus visitantes para matérias antigas abordadas no site.
No admin é possível configurar 3 opções:
- Número máximo de thumbnails a exibir;
- Quantos thumbnails por linha;
- Dimensão do thumbnail.

== Installation ==

1. Descompacte o arquivo `random-thumbs.zip` no diretório `/wp-content/plugins/`;
1. Ative o plugin no item 'Plugins' do menu do Wordpress;
1. Insira `[randomthumbs]` dentro de um POST ou PAGE.

== Frequently Asked Questions ==
Q: Como exibo os thumbnails?
A: Insira em seu post o seguinte '[randomthumbs]'.
